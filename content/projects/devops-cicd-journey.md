## The DevOps Lab

Infrastructure reliability is just as vital as clean code. This repository logs our hero's journey in implementing modern DevOps practices, focusing on automation, testing, and continuous deployments.

### Labs Implemented

* **GitHub Actions Workflows**: Auto-trigger tests, linting runs, and docker builds on every branch commit.
* **Linux Server Hardening**: Automating deployment parameters onto remote VPS targets.
* **Docker Container Registry Sync**: Auto-publishing container packages to Docker Hub registries.
