## The Vision

Standard typing tools force you to practice on boring predefined dictionary paragraphs. **TypeFlow** lets users paste their own customized text (like lecture notes, code files, or foreign language texts) and immediately generates responsive, visual typing sessions.

### Features

* **Dynamic Text Tokenizer**: Splits user copy into readable lines and word arrays with real-time cursor bounds.
* **Accuracy Tracking**: Measures raw Words Per Minute (WPM), Net WPM, and key-by-key accuracy charts.
* **Theme Modes**: Retro print design palettes that react to typing milestones.
* **Local Stats Logs**: Persists historical speed logs to local storage, mapping performance over time.
